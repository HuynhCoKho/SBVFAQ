Bạn nhập câu hỏi về quy định. Hệ thống sẽ gửi câu hỏi sang Apps Script, đọc dữ liệu Google Sheet và dùng AI để trả lời.

Trang public

  1. Giao diện này public bằng GitHub Pages.
  2. Backend là Apps Script Web App.
  3. Dữ liệu lấy từ Google Sheet FAQ và VANBAN.
  4. Lịch sử hỏi đáp được ghi vào sheet LOG.

Nếu trang báo chưa cấu hình, hãy dán URL Apps Script kết thúc bằng /exec vào file config.js.
